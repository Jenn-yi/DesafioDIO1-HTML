Inspetor de elementos

O Inspetor de Elementos é uma função embutida nos navegadores, e através dele é possível ver como o HTML foi utilizado numa página e também sua estilização.

Estrutura básica do HTML

Começando pela estrutura básica do HTML, uma página possui três elementos principais: html, head e body.

HTML: É o que faz o navegador interpretar que se trata de um código HTML.
HEAD: É a cabeça do site, onde ficam suas informações principais que não são exibidas para o usuário, como as tags meta que são responsáveis por auxiliar no algoritmo.
BODY: É o corpo do site, onde fica o conteúdo que será exibido para o usuário, como textos e imagens.

Atributos das Tags

As tags possuem atributos, que são capazes de alterar a propriedade delas. Embora o objetivo da tag continue o mesmo, os atributos modelam as tags para que fique mais de acordo com o seu propósito. Alguns exemplos:

ID: Adiciona um identificador a uma tag, útil para aplicação de estilos em CSS e também para referenciá-las no caso da aplicação de links.
CLASS: Enquanto o ID funciona para apenas uma tag, o CLASS permite adicionar identificador a mais de um elemento ao mesmo tempo, como títulos e parágrafos.
SRC: Utilizada na tag IMG, é colocado o diretório da imagem utilizada, que pode ser um URL da Internet ou do próprio computador, se colocado na mesma pasta do código HTML.

Links

Através de links, é possível direcionar o usuário para outros sites, ou navegar dentro do próprio site.

Para isso, é utilizado o comando "a", que se refere a ancoragem (ou seja, faz o usuário ancorar/ir para um lugar). E, o href, assim como o src é utilizado no caso das imagens, é colocado dentro do a (sendo um atributo da tag) para que seja definido a URL.
